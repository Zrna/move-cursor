A script that moves your cursor automatically.

The initial duration is `1 minute`.

## Usage

---

Instal using `npm` (globally):

```sh
npm install move-cursor -g
```

Run command from the terminal:

```sh
move-cursor 10 sec
```

---

Run using `npx`:

```sh
npx move-cursor 10 sec
```
